# Tax Calculation App!

## Prerequisite

 - .NET 7 SDK and runtime

## Build and Run Application

 - Make sure all Nuget packages are restored.
 - Build solution
 - Run application with Multiple Startup
 - ***TaxCalculations.API and TaxCalculations.ClientApp***

# Database Files
TaxCalculation.mdf
TaxCalculation.log
